# 🌍 Voyage Aid AI  
### *Su planificador de viajes personal con IA*  

Voyage Aid AI es un generador inteligente de itinerarios de viaje que crea **planes de viaje personalizados** para los usuarios en función de sus preferencias, el tiempo disponible y el destino.

Aprovecha la **IA**, **Maps** y **YouTube** para diseñar rutas de viaje inteligentes, eficientes y apasionantes, que incluyen paradas gastronómicas, sugerencias de hoteles y joyas locales poco conocidas.  

---

## 🚀 Características  

✅ **Itinerarios impulsados ​​por IA** – Genera planes de viaje personalizados según las preferencias del usuario y el tiempo disponible.  
✅ **Integración de mapas** – Proporciona indicaciones, tiempos de viaje y enlaces de ubicación directamente en el itinerario.  
✅ **Integración con YouTube** – Incluye vlogs de viajes mejor valorados para cada destino.  
✅ **Planificación inteligente de la alimentación** – Incluye automáticamente pausas para el desayuno, el almuerzo y la cena; hay pausas opcionales para refrigerios.  
✅ **Cuentas de usuario** – Inicia sesión o regístrate para guardar y recibir itinerarios por correo electrónico.  
✅ **Integración de correo electrónico** – Los usuarios pueden recibir una copia de su itinerario personalizado por correo electrónico.  
✅ **Interfaz de usuario hermosa y estética** – Interfaz limpia de colores pastel, con formas flotantes y transiciones modernas.  

---

## 🧠 Cómo funciona  

1. **Regístrate o inicia sesión**  
   Los usuarios crean una cuenta con su correo electrónico y contraseña (cifrada en el lado del cliente).  

2. **Acceder a las preferencias**  
   Elige la ciudad, la duración del viaje y tus intereses (historia, gastronomía, aventura, etc.).  

3. **Generación de itinerarios con IA**  
   Voyage Aid AI utiliza datos previamente cargados y API de IA gratuitas para generar planes de viaje optimizados.  

4. **Explorar y modificar**  
   Los usuarios pueden ver rutas, vídeos de YouTube y servicios cercanos, e incluso sustituir aquellos lugares que prefieran omitir.  

5. **Recibir por correo electrónico**  
   El itinerario definitivo puede enviarse directamente por correo electrónico al usuario a través de **EmailJS**.

## 🧩 Stack Tecnológico  

| Capa | Tecnología |
|-------|-------------|
| **Frontend** | HTML, CSS, JavaScript |
| **Estilo** | CSS3, Degradados pastel, Glassmorfismo |
| **Motor de IA** | OpenRouter API |
| **Mapas y rutas** | Open Route Service + OpenTrip Map |
| **Envío de correos electrónicos** | EmailJS |
| **Alojamiento** | [voyage-aid-ai.w3spaces.com](https://voyage-aid-ai.w3spaces.com) |

---

## 🔑 API utilizadas  

| Propósito | API |
|----------|-----|
| Mapas y rutas | [OpenTrip Map] y [Open Route Service] |
| Envío de correos electrónicos | [EmailJS](https://www.emailjs.com/) |
| Generación de texto mediante IA | [OpenRouter](https://www.openrouter.ai/) |

---

## 🔐 Notas de seguridad  

- Las contraseñas se someten a hash utilizando **bcrypt.js** antes de almacenarse.  
- Los correos electrónicos se almacenan en texto plano (solo para la entrega del correo).  
- No se recopilan datos personales sensibles.
