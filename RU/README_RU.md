# 🌍 Voyage Aid AI
### *Ваш персональный планировщик путешествий на базе искусственного интеллекта*  

Voyage Aid AI (Вояж Эйд Эй-Ай) — это интеллектуальный генератор маршрутов, который создает **персонализированные планы поездок** для пользователей с учетом их предпочтений, временных рамок и пункта назначения.  
Сервис использует возможности **ИИ**, **Карт** и **Ютуб** для создания продуманных, эффективных и увлекательных маршрутов, включающих места, где можно перекусить, варианты размещения и интересные, но малоизвестные локальные достопримечательности.  

---

## 🚀 Функции  

✅ **Маршруты, созданные с помощью ИИ** – Создает индивидуальные планы поездок с учетом предпочтений пользователя и имеющегося времени.  
✅ **Интеграция с картами** – Предоставляет маршруты, время в пути и ссылки на местоположение прямо в плане поездки.  
✅ **Интеграция с Ютуб** – Прилагает лучшие тревел-влоги для каждого направления.  
✅ **Разумное планирование питания** – Автоматически включает перерывы на завтрак, обед и ужин; также предусмотрена возможность делать перерывы на перекус.  
✅ **Учетные записи пользователей** – Войдите или зарегистрируйтесь, чтобы сохранить маршруты и получать их по электронной почте.  
✅ **Интеграция с электронной почтой** – Пользователи могут получить копию своего индивидуального маршрута по электронной почте.  
✅ **Красивый и эстетичный интерфейс** – Чистый интерфейс в пастельных тонах с «плавающими» фигурами и современными переходами.  

---

## 🧠 Как это работает  

1. **Зарегистрироваться или войти**  
   Пользователи создают учетную запись, используя адрес электронной почты и пароль (шифруемый на стороне клиента).  

2. **Перейти к настройкам**  
   Выберите город, продолжительность поездки и интересы (история, гастрономия, приключения и т. д.).  

3. **Генерация маршрутов с помощью ИИ**  
   Voyage Aid AI использует предварительно загруженные данные и бесплатные API искусственного интеллекта для создания оптимизированных планов путешествий.  

4. **Исследуйте и изменяйте**  
   Пользователи могут просматривать маршруты, видео на YouTube и информацию об объектах инфраструктуры поблизости, а также заменять места, которые предпочли бы пропустить.  

5. **Получить по электронной почте**  
   Окончательный маршрут может быть отправлен пользователю по электронной почте напрямую через **EmailJS**.

## 🔐 Замечания по безопасности  

- Перед сохранением пароли хешируются с помощью **bcrypt.js**.  
- Электронные письма хранятся в виде открытого текста (только для целей доставки).  
- Конфиденциальные персональные данные не собираются.
